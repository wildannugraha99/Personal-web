How i write a code?

Menulis code yang bersih

Code yang baik bukan hanya berjalan sesuai fungsinya, tapi juga mudah dibaca oleh penulisnya. Kode yang bersih dan mudah dibaca, akan mudah diperbaiki ketika ada bug dan akan mudah diupdate ketika ada fungsi tambahan.

Menulis code yang mudah di baca

Code yaang mudah di pahami dan mudah di baca oleh semua orang adalah salah satu syarat yang harus di miliki oleh seorang programmer.

Dokumentasi yang rapi

Programmer yang baik bukan hanya programmer yang dapat membuat code atau aplikasi yang baik dan dapat berjalan, tapi juga harus memiliki dokumentasi yang baik dan rapi agar orang lain mudah mengerti dengan code yang di buat.
